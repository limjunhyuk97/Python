# Mutable, Immutable 객체
  - Mutable : list, dictionary, set
  - Immutable : number, string, tuple
 
## 01. Mutable
  - 한번 선언된 이후로 값이 변할 수 있는 자료형
  - 변수가 참조하고 있는 객체의 값이 변할 수 있다
  - List, Dictionary, set

## 02. Immutable
  - 한번 선언되 이후로는 값이 변할 수 없는 자료형
  - 변수가 가리키는 값의 변경은, 객체의 변경이 아니라, 새로 생성된 객체의 할당이다.
  - Number, String, Tuple
